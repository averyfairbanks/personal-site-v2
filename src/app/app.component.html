About Me

Hailing originally from New England, Avery is a developer - specifically focused on full stack web design & development - whose main expertise lies in Angular web framework and Java SpringBoot. However, they remain a multidisciplinary programmer and continue to learn new languages and frameworks.

Tasked with creating a full stack web application for handling checkouts at the food pantry. I handled large portions of the Java SpringBoot Rest API as well as designing the UI & UX using Adobe Xd and implementing it with Angular.

I wrote numerous Angular animations for window transitions as well as utilizing Form Validators, Services, Emitters, and tons more tools that the framework has to offer. This was easily the largest and most ambitious project I have worked on but I look forward to surpassing it.

This very site is the third revision of my personal site, which began as a project for my Software Construction course. All three versions have been built in Angular, integrating new features and technologies -- like Material, LESS/SASS, and many others -- as I learned them.

While not exhaustive, this site represents the culmination of my relevant design and development skills. In addition, I have built numerous concept sites exploring content display, all of which can be found on my github page here.
